What is Beta Testing?

A beta release of software is the phase in which a product includes all of the product’s functionality but it has not been tested by external users. The DOwhit.com website has been tested in our own development environment, but it hasn’t been tested on other computers. You, as a tester, will help us identify if there are problems with accessing the website using different operating systems and internet connections.

When beta software is released to a group of people, called beta testers, they use it and then report anything they find which needs to be corrected. You, as a beta tester, will help us identify if there are configuration problems with the software and the functionality of the DOwhit.com website. Your feedback on any problems you find, as well as any suggestions you can provide, will help us identify any problems before we launch our website.

What do I need to do for testing?

The DOwhit.com website doesn’t require any software installation. It works in a browser on both Windows and Macintosh computers. You can use the following browsers to test the website:

  • Google Chrome (for Windows and Mac) **Preferred** Download
  • Microsoft Internet Explorer (IE) 8.0 and later (for Windows) Download
  • Firefox 3.6 and later (for Windows, Mac, and Linux) Download
  • Safari 3.1 and later (for Mac and Windows) Download

What do I do if I have a problem using the website, DOwhit.com?

Take careful note of exactly what occurred when you find a problem. We will need to know if any error messages were displayed and details of exactly what you were doing on your computer when the problem occurred. This will help us reproduce and correct the problem.

To submit feedback or report errors, please use the feedback tab on the right side of the site or email us at feedback@dowhit.com.
